## Getting Started

1. Set
   ```bash
   yarn setup
   ```

   1. Install packages
   1. Run migration
   1. Run seeder

1. Run the development server
   ```bash
   yarn start
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
